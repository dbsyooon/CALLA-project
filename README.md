# **CALLA**

**CALLA**은 *국민건강영양조사(KNHANES)* 데이터를 활용해
사용자의 건강 정보를 분석하고, AI 기반 자연어 입력을 통해 통계적 비교·건강 인사이트를 제공하는 건강 분석 웹 애플리케이션입니다.
Calla AI 엔진이 사용자의 텍스트 입력을 이해하고 건강 지표를 자동 추출하여 연령·성별 그룹 기반의 통계 분석과 질병 위험도를 제공합니다.

---

## 🚀 **주요 기능**

### 1. **자연어 기반 건강 분석**

사용자가 자연어로 입력한 문장에서 건강 정보를 자동 추출합니다.

예시 입력
```
26살 남성 당화혈색소 6.9
```

추출되는 정보:
* **나이**: 정규표현식 기반 추출
* **성별**: 남성 / 여성
* **측정값**: 숫자
* **건강 지표명 매칭**: Sentence Transformers 기반 유사도 분석
  (KR-SBERT-V40K-klueNLI-augSTS)

---

### 2. **통계적 분석**

국민건강영양조사(KNHANES) 데이터를 기준으로
동일 연령대·성별 집단과 비교 분석을 수행합니다.

제공 지표:
* Z-score 기반 위치 분석
* 분위수 (25%, 50%, 75%)
* ±1σ, ±2σ 범위
* 집단 대비 백분위 분석

---

### 3. **질병 위험도 평가**

주요 만성질환에 대한 위험도를 규칙 기반으로 평가합니다.

| 질환       | 기준                                     |
| -------- | -------------------------------------- |
| **당뇨병**  | 공복혈당 ≥ 126 mg/dL<br>당화혈색소 ≥ 6.5%       |
| **고지혈증** | 총콜레스테롤 ≥ 240 mg/dL<br>중성지방 ≥ 200 mg/dL |
| **고혈압**  | 수축기 ≥ 140 mmHg<br>이완기 ≥ 90 mmHg        |
| **통풍**   | 요산 ≥ 7.0 mg/dL                         |
| **빈혈**   | 헤모글로빈 < 12.0 g/dL                      |

---

### 4. **대시보드**

사용자의 건강 정보를 시각적으로 제공합니다.
* 개인 건강 요약 카드
* 주요 질병 위험도
* 향후 클러스터 분석 기능 제공 예정

## 🛠️ **기술 스택**

### **Backend**

* **FastAPI**
* **SQLite + SQLAlchemy ORM**
* **인증**: JWT (python-jose), bcrypt
* **AI/ML**

  * Sentence Transformers (KR-SBERT-V40K-klueNLI-augSTS)
  * PyTorch
  * soynlp (Korean tokenizer)
* **Data 처리**: pandas, numpy

### **Frontend**

* **React 18 + TypeScript**
* **Vite**
* **React Router v7**
* **Tailwind CSS**
* **Radix UI 기반 커스텀 컴포넌트**
* **Lucide Icons**

---

## 📊 **기술 개요 다이어그램**

```
사용자 입력 → 자연어 처리(SBERT) → 지표 매칭  
            ↓  
      나이/성별 추출  
            ↓  
  KNHANES 통계 비교 → Z-score / 분위수 계산  
            ↓  
    질병 위험도 평가  
            ↓  
      대시보드 시각화
```
